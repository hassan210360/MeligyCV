FEVERISH - Bold Home Decor Site

New "Feverish" Template Features:
- Hot color scheme: #FF3D00 orange, #FFD600 yellow, #0A0A0A black
- Bebas Neue + Montserrat fonts for aggressive branding
- Fixed header with cart icon + live item count
- Full shopping cart sidebar with add/remove items
- WhatsApp checkout: sends full cart to your number
- Individual WhatsApp buttons per product
- Product loop: assets/product1.jpg to product20.jpg
- Badges for first 5 items marked "NEW"
- Mobile responsive + slide-out cart

Setup:
1. Replace assets/logo.png, banner.jpg, product1-20.jpg
2. Update WhatsApp: search '201234567890' in index.html
3. Update Instagram/Facebook: search 'yourbrand'
4. Open index.html - no build needed

Cart stores in browser only. For real payments, you'd need backend integration.
